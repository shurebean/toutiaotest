# 今日头条热点抓取系统 - 详细开发计划

## 项目基本信息

- **项目名称**: 今日头条热点抓取系统
- **项目目录**: `/home/toutiaotest`
- **开发周期**: 预计 2-3 周
- **技术栈**: 
  - 后端: Node.js + Express + Puppeteer + MongoDB
  - 前端: Vue 3 + Vite + Element Plus
  - AI: OpenAI API
  - 部署: Docker + Docker Compose
